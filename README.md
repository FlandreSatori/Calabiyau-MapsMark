在 Vercel 项目设置中添加以下环境变量：

- `GITHUB_OWNER`
- `GITHUB_REPO`
- `GITHUB_BRANCH`
- `GITHUB_DATA_PATH`
- `GITHUB_TOKEN`
- `ADMIN_TOKEN`

然后部署到 Vercel 即可。

用 iframe 嵌入：

```html
<iframe
  src="https://your-domain.vercel.app/embed?bg=rgba(8,12,22,0.45)"
  style="width:100%;height:900px;border:0;border-radius:20px;overflow:hidden;"
  loading="lazy"
></iframe>
```
